This is an app which recognizes handwritten digits. It is a deep learning based project in which first a model is trained and then it is set for predicting the data based on what it has learned.

In this app we have used tensorflow framework. Google colab is used for python coding and model training. visual studio code is used for HTML,CSS and Java Script. This app is deployed using Heroku. Some other libraries like keras ,numpy and Scikit-learn are also used.
